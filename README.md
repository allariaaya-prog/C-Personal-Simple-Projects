# C-Personal-Simple-Projects & problems solving
This repository contains small personal projects & problems solving I created to practice C++ programming and strengthen my coding skills.

## Projects Included
- **Calculator**: A simple calculator supporting basic arithmetic operations.
- **Guess the Number**: Number guessing game to practice conditional statements and random numbers.
- **Clock / Timer**: A program that displays real-time clock or counts down/up as a timer, practicing loops, functions, and time manipulation.
- **ATM Interface**: Simulates an ATM screen with options like balance check, deposit, and withdraw, practicing menus, user input, and logic flow.
- **Problem_Solving**: A collection of problem solving in different topics in this language.
  

  ## Technologies Used
- C++ (Standard Library, STL where applicable)

  ## Notes
- Each project is contained in its own folder with source code and instructions.
